# gazprombank-tech-blog-publications
Responsive multi-page blog website consisting of **Blog** and **Article** pages with navigation between them.

## Description
This project is a responsive layout for an IT blog platform. It includes a blog page with article cards and a detailed article page. Smooth animations and interactive elements are implemented to enhance user experience.


## Technologies
- HTML5
- CSS3 / SCSS
- Responsive layout (Flexbox / Grid)
- BEM methodology
- 
## Project description
- Multi-page structure (Blog & Article pages)
- Navigation between pages
- Fully responsive layout
- Semantic HTML markup
- CSS animations and hover effects
- Interactive article cards:
  - Image smoothly shifts downward on hover/click
- Clean and well-structured codebase
